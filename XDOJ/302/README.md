# 质数求和

## 时间限制：

1S

## 问题描述	

输入一个大于20的正整数p，求出正整数范围内**第p个质数**(或素数，大于1且只能被1和自己整除的正整数,)到**第p+10个质数**之**和**。其中，第1个质数为2，第2个质数为3，第3个质数为5，第4个质数为7，依次类推。

## 输入说明	

一行输入1个整数p，20< p <150。

## 输出说明	

第p个质数到第p+10个质数**之和**。

## 输入样例：

```
21
```

## 输出样例：

````
1081
```
